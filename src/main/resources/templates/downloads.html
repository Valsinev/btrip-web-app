За да потръгне обаче трябва да имате инсталирана Java(версия 17 или по-нова) на компютъра.

Начин на инсталиране:

Може да си изтеглите Java от тук.

Изберете вида операционна система.

Изберете файлът отговарящ на типът операционна система

и го инсталирайте.

Направете програмата изпълняема(executable):

За Windows операционна система:

Ако след инсталиране на Java програмата не тръгва за съжаление трябва да потърсите инструкции за подкарване на Jar файл на windows.

Начините са няколко и са дълги за обяснение.

За Linux операционна система:

Щракнете на файлът с дясно копче.

Изберете: Oпции(Properties).

От подменюто - Позволения(Permissions) сложете отметката на:

позволете изпълнението на файла като програма.

(Allow executing file as program)
